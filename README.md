### Set-up
1. Set up the working environment.
   1. Install a package manager: `Node.js` and `npm`.
   2. Install `Three.js` for graphics
   3. Install `vite` to run your local web server.

```bash
npm init -y
npm install --save three
npm install --save-dev vite
```

### Running 

```bash
npx vite
```
